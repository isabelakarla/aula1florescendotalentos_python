## Lógica de Programação | Turma 1 — EREM Rosa de Magalhães Melo

Bem-vindo(a) ao repositório da nossa **Primeira Aula** de Lógica de Programação! Este espaço foi criado para guardar o código que desenvolvemos juntos e servir de apoio para os seus estudos.

---

📌 Sobre a Aula
Nesta aula introdutória para a **Turma 1 (1º Ano do Ensino Médio)**, nós demos os primeiros passos no mundo do Python. O grande objetivo foi entender como o computador "pensa" (lógica de programação) e como podemos organizar nosso código de forma inteligente.

Desenvolvemos juntos uma **Calculadora Funcional** utilizando conceitos fundamentais da linguagem.

---

🚀 Conceitos Aprendidos
Durante a construção do projeto, nós exploramos:
* **Variáveis e Tipos de Dados:** Como armazenar números e textos na memória.
* **Funções (`def`):** Como criar blocos de código reutilizáveis para organizar nossa calculadora (uma função para cada operação matemática).
* **Estruturas Condicionais (`if`, `elif`, `else`):** Como fazer o programa tomar decisões com base na escolha do usuário.
* **Entrada e Saída de Dados (`input` e `print`):** Como interagir com o usuário, recebendo comandos e exibindo os resultados na tela.

---

🧮 O Projeto: Calculadora em Python
O código final é capaz de realizar as quatro operações matemáticas básicas:
1. Soma (`+`)
2. Subtração (`-`)
3. Multiplicação (`*`)
4. Divisão (`/`)

Como executar o código:
1. Como não é possível termos uma IDE como o VS Code, acesse online o site "www.gdbonline.com.br"
2. Selecione "Python 3" na seção de linguagem.
3. Copie e cole o código da calculadora no espaço indicado
4. Por fim, clique em "RUN" para rodar o código
